# Battleship Royale Game

## 📌 Description

Battleship Royale is a console-based strategy game developed as my Semester 1 project.
The game is inspired by the classic Battleship game, where players attempt to locate and destroy enemy ships.

## ⚙️ Features

* Grid-based gameplay
* Player vs Player interaction
* Ship placement and tracking
* Hit/miss detection system
* Turn-based gameplay

## 💻 Language Used

* C++ 

## 📚 Concepts Used

* Arrays / 2D Arrays
* Loops and Conditions
* Functions
* Game logic implementation
* Problem-solving and algorithmic thinking

## ▶️ How to Run

1. Open the project in your IDE
2. Compile the code
3. Run the program

## 🎯 Purpose

This project was developed to:

* Apply programming fundamentals in a real project
* Build a complete game from scratch
* Strengthen problem-solving and logical thinking
